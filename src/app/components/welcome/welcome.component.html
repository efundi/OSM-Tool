Welcome to PDF Marker.

The onscreen PDF marking application. If you have just installed the application, please click settings above and configure your application.

Application Icons and Tools
unarchive Import an assignment from a zip file for marking.
picture_as_pdf Upload PDFs and create an assignment for marking.
settings Application Settings and Rubric Settings.
Assignment Overview Settings

Access the assignment overview by clicking on the assignment name.

apps View and change selected rubric.
archive Finalize and export assignment to zip.
person Add or remove students from a PDF Marker created assignment.
Marking Icons and Tools

Assessors can either mark with a rubric or with a set of marking tools as listed below. Rubrics are enabled when you import or create an assignment.

check Single Mark or Tick, has a value of 1.
halfTick Half-mark or half-tick, has a value of 0.5.
spellcheck Acknowledge mark. This marking tool has no mark value assigned to it.
close Cross mark used for indicating a wrong answer.
comment Comment or number mark. This marking tool allows you to assign a mark of any number to a section. There is space for an optional comment.
announcement Preview Marks will show you the current breakdown of marks on the visible PDF and a breakdown of what marking tools these come from.
arrow_upward Page the pdf upwards.
arrow_downward Page the pdf downwards.
layout-expand-left Show left panel (PDF) only when marking with a rubric.
layout-default Show both panels (PDF and Rubric) when marking with a rubric.
layout-expand-right Show right panel (Rubric) only when marking with a rubric.
palette Change marking icon color, all mark icons from now on will use this color.
delete_sweep Clear all marks, selections, comments and annotations.
