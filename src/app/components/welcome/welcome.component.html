Welcome to PDF Marker.

The onscreen PDF marking application. If you have just installed the application, please navigate to App Settings and configure your application.

Application Icons and Tools
assignment_outlined Working folders and assignments
unarchive Import an assignment from a zip file for marking
assignment_add Create a manual assignment for marking by capturing student submissions
settings Application settings, rubrics, generic comments and user management
Assignment Overview Settings

Access the assignment overview by clicking on the assignment name.

apps View and change selected rubric
person Add or remove students from a manually created assignment
workspaces Allocate or re-allocate for marking (for users managing assessments)
unarchive Import marked work back from markers (for users managing assessments)
verified Moderation and moderation verification
archive Export for Review (for markers) and Finalise and Export (for users managing assessments)
Marking Icons and Tools
check Single Mark or Tick, has a value of 1
halfTick Half-mark or half-tick, has a value of 0.5
spellcheck Acknowledge mark. This marking tool has no mark value assigned to it
close Cross mark used for indicating a wrong answer
comment Comment and/or mark tool
brush Highlight tool with optional comment
layout-expand-left Show left panel (PDF) only when marking with a rubric
layout-default Show both panels (PDF and Rubric) when marking with a rubric
layout-expand-right Show right panel (Rubric) only when marking with a rubric
palette Change marking icon color, all mark icons from now on will use this color
delete_sweep Clear all marks, selections, comments and annotations
